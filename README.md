## Creando mi primer documento HTML

1. Abre tu editor de código
2. Crea un nuevo espacio de trabajo (carpeta o proyecto)
3. Crea un nuevo archivo y llámalo index.html
4. Crea la estructura básica de un documento HTML
5. Añade un título Mis comidas favoritas en el cuerpo usando elementos Encabezados HTML
6. Añade una lista de tus alimentos favoritos usando elementos Listas en HTML
7. Ejecuta la página web para ver el resultado final (Sí estas en VS Code puedes usar la extensión Live Server)
